# 🎓 无格式论文写作器 - 用户使用手册

## 快速开始

### 1. 安装和启动
```bash
# 安装依赖
npm install

# 启动应用
npm run dev
```

### 2. 创建第一个项目
1. 点击"新建论文"按钮
2. 输入论文标题
3. 系统自动创建默认章节结构

### 3. 编辑内容
- 在左侧大纲中选择章节
- 在中间编辑器中编写内容
- 右侧实时查看预览

## 核心功能使用

### 📝 章节管理
- **添加章节**: 点击"新建章节"
- **重命名**: 双击章节标题
- **删除**: 右键菜单选择删除
- **排序**: 拖拽章节调整顺序

### 🖼️ 图片管理
1. 点击工具栏"插入图片"
2. 选择图片文件
3. 输入图片标题
4. 自动编号(如:图1-1)

### 📚 参考文献管理

#### 手动添加
1. 打开参考文献面板
2. 点击"添加"
3. 填写文献信息

#### BibTeX导入
1. 点击"导入"按钮
2. 选择.bib文件或粘贴内容
3. 预览解析结果
4. 确认导入

#### 插入引用
1. 光标定位到插入位置
2. 点击"插入引用"
3. 选择文献
4. 自动生成引用标记

### 💾 自动保存
- 系统每30秒自动保存
- 状态栏显示"未保存更改"提示
- 可手动点击"保存"按钮

### 📤 导出Word文档
1. 点击"导出 Word"按钮
2. 选择保存位置
3. 等待生成完成
4. 打开.docx文件查看

## 高级功能

### 模板设置
1. 点击"格式设置"
2. 选择学校模板
3. 自定义格式参数:

   - 字体设置(各级别标题、正文)
   - 段落设置(行距、缩进)

### 项目管理
1. 点击"打开项目"
2. 查看所有项目列表
3. 选择要打开的项目
4. 自动加载所有内容

### 实时预览
- 右侧预览面板实时显示
- 应用模板格式
- 自动更新
- 可缩放查看

## 常见问题

### Q: 数据保存在哪里?
**A:** 数据保存在本地SQLite数据库中
- macOS: `~/Library/Application Support/无格式论文写作器/thesis-writer.db`
- Windows: `%APPDATA%/无格式论文写作器/thesis-writer.db`
- Linux: `~/.config/无格式论文写作器/thesis-writer.db`

### Q: 如何备份数据?
**A:** 复制上述数据库文件即可

### Q: 导出的Word格式不对?
**A:**
1. 检查模板设置是否正确
2. 确认章节内容格式规范
3. 查看参考文献格式设置

### Q: 图片无法显示?
**A:**
1. 检查图片路径是否正确
2. 确认图片格式支持(JPG/PNG)
3. 重新插入图片

### Q: BibTeX导入失败?
**A:**
1. 检查BibTeX格式是否正确
2. 确认文件编码为UTF-8
3. 尝试手动粘贴内容

## 最佳实践

### 写作流程建议
1. **规划结构** - 先创建完整的章节大纲
2. **选择模板** - 确定目标学校格式要求
3. **分章写作** - 按章节顺序编写内容
4. **插入图表** - 添加图片和表格
5. **管理文献** - 导入并整理参考文献
6. **插入引用** - 在正文中添加引用
7. **预览检查** - 查看实时预览效果
8. **导出验证** - 导出Word并检查格式
9. **最终修改** - 根据导出结果调整

### 文献管理建议
- 使用Zotero/EndNote导出BibTeX
- 定期整理文献库
- 检查引用格式
- 删除重复文献

### 格式设置建议
- 参考学校论文要求
- 测试导出效果
- 保存常用模板
- 备份模板配置
