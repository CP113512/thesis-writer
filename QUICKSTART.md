# 快速开始指南

## 数据库功能已完成 ✅

无格式论文写作器现在已经支持完整的数据持久化功能!

### 测试步骤

1. **打开应用**
   - 应用已在运行中
   - 如果没有,运行: `npm run dev`

2. **创建新项目**
   - 点击"新建论文"按钮
   - 输入论文标题,例如:"基于深度学习的图像识别研究"
   - 点击"创建"

3. **查看项目保存**
   - 项目会自动保存到数据库
   - 点击"保存"按钮手动保存

4. **测试项目加载**
   - 点击"打开项目"按钮
   - 查看已保存的项目列表
   - 点击任意项目加载

### 数据库位置

~/Library/Application Support/无格式论文写作器/thesis-writer.db

### 技术栈

- **前端**: React 19 + TypeScript + Vite
- **状态管理**: Zustand
- **编辑器**: TipTap
- **数据库**: sql.js (SQLite)
- **桌面框架**: Electron 28
- **样式**: CSS (可升级到Tailwind)

### 问题排查

如果遇到问题:

1. **数据库初始化失败**
   - 检查数据库文件权限
   - 删除数据库文件重新初始化

2. **项目无法保存**
   - 查看控制台错误信息
   - 检查数据库文件是否锁定

3. **应用无法启动**
   - 运行 `npm run build:electron`
   - 删除 `node_modules` 重新安装
