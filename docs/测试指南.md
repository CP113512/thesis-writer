# 论文写作器功能测试指南

## 测试方法

### 方法一：浏览器开发者工具测试

1. 启动应用：`npm run dev`
2. 打开应用窗口后，按 `Cmd+Option+I`（Mac）或 `Ctrl+Shift+I`（Windows）打开开发者工具
3. 在 Console 标签中复制粘贴 `test-all-features.js` 的内容
4. 运行命令：`runAllTests()`
5. 查看测试结果

## 常见问题排查

### 问题1：数据库初始化失败
- 检查 Electron API 是否正常
- 查看控制台错误信息

### 问题2：导出失败
- 检查模板配置是否存在
- 查看导出日志 `/tmp/thesis-export.log`

### 问题3：图片不显示
- 检查图片是否为 base64 格式
- 检查图片大小是否过大

### 问题4：引用显示异常
- 检查参考文献 ID 是否正确
- 检查扩展是否正确注册

## 自动化测试脚本

完整测试脚本位于 `test-all-features.js`，可在浏览器控制台运行。

运行方式：
1. 打开开发者工具（Cmd+Option+I）
2. 复制 `test-all-features.js` 内容到控制台
3. 执行 `runAllTests()`
4. 查看测试报告
