## 🏗️ 架构设计

### 分层架构
```
┌─────────────────────────────────┐
│        用户界面层 (React)         │
│  Sidebar | Editor | Preview      │
├─────────────────────────────────┤
│        业务逻辑层 (Zustand)       │
│  Project | Editor | Reference    │
├─────────────────────────────────┤
│        服务层 (Services)          │
│  API | Export | Numbering        │
├─────────────────────────────────┤
│        IPC通信层 (Electron)       │
│  Database | Export | FileDialog  │
├─────────────────────────────────┤

│  Projects | Chapters | Refs      │
└─────────────────────────────────┘
```

### 数据流
```
用户操作 → React组件 → Zustand Store → 
API Service → IPC → Electron Main → 
Database Service → SQLite
```
